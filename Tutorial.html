Descubre cómo automatizar el riego de tus plantas con Arduino y aplicación móvil.

Tutorial de Instalación

Este tutorial te guiará paso a paso en la construcción del sistema de riego automático con Arduino y la instalación de la app móvil para su control.

Código para Arduino


#include <SoftwareSerial.h>

SoftwareSerial BT(10, 11);  // RX, TX para Bluetooth

int bomba = 8;
int humedad = 0;
bool modoManual = false;  // Control manual o automático

void setup() {
  Serial.begin(9600);
  BT.begin(9600);

  pinMode(bomba, OUTPUT);
  digitalWrite(bomba, HIGH);  // La bomba inicia apagada
}

void loop() {
  if (BT.available()) {
    String comando = BT.readString();
    comando.trim();

    if (comando == "MANUAL") {
      modoManual = true;
      Serial.println("Modo manual activado");
    } else if (comando == "AUTO") {
      modoManual = false;
      Serial.println("Modo automático activado");
    } else if (modoManual) {
      if (comando == "ON") {
        digitalWrite(bomba, LOW);
        Serial.println("Bomba ENCENDIDA (manual)");
      } else if (comando == "OFF") {
        digitalWrite(bomba, HIGH);
        Serial.println("Bomba APAGADA (manual)");
      }
    }
  }

  if (!modoManual) {
    humedad = analogRead(A0);

    if (humedad >= 721 && humedad <= 1024) {
      digitalWrite(bomba, LOW);
      Serial.println("Bomba ENCENDIDA (automático)");
    } else {
      digitalWrite(bomba, HIGH);
      Serial.println("Bomba APAGADA (automático)");
    }

    Serial.println(humedad);

    BT.print(humedad);  
    BT.print("\r");     

    delay(500);
  }
}

Construcción y Armado de componentes

1. Armado del Sistema en Arduino

Materiales Necesarios

  • Sensor de humedad:
    • VCC → 5V de Arduino
    • GND → GND de Arduino
    • A0 → Pin A0 de Arduino
  • Módulo Bluetooth HC-05/HC-06:
    • VCC → 5V de Arduino
    • GND → GND de Arduino
    • TX → Pin 10 de Arduino
    • RX → Pin 11 de Arduino (con divisor de voltaje 1K y 2K)
  • Bomba de agua y relé:
    • IN del relé → Pin 7 de Arduino
    • VCC del relé → 5V de Arduino
    • GND del relé → GND de Arduino
    • COM del relé → GND de la bomba
    • NO del relé → VCC de la bomba

Nota: Si la bomba es de 12V, usa una fuente externa.

      1. Instalación y Requisitos Previos

      • Tener la aplicación instalada en tu dispositivo móvil. Descarga
      • Contar con un módulo Bluetooth en el sistema de riego.
      • Tener activado el Bluetooth en tu celular.

      3. Conexión con el Sistema de Riego

      • Asegúrate de que el módulo Bluetooth del sistema de riego esté encendido.
      • En la aplicación, presiona el botón de "Conectar Bluetooth".
      • Aparecerá una lista de dispositivos disponibles, selecciona el módulo de riego.
      • Espera la confirmación de conexión.

      4. Monitoreo de la Humedad

      • En la pantalla principal de la app, verás un indicador con el porcentaje de humedad actual del suelo.
      • Si el porcentaje de humedad es bajo, la app puede recomendar activar el riego.
      • Se actualiza en tiempo real conforme la humedad cambia.

      5. Modos de Funcionamiento

      Modo Automático

      • Si activas esta opción, el sistema encenderá o apagará el motor automáticamente con base en los valores de humedad.
      • Ideal para mantener un riego eficiente sin intervención manual.

      Cómo activarlo:

      • Presiona el botón "Automático" en la aplicación.
      • Verás una confirmación de que el sistema manejará el riego según la humedad.

      Modo Manual

      • En este modo, tú decides cuándo encender o apagar el motor.
      • Útil si quieres un control más preciso del riego.

      Cómo usarlo:

      • Presiona el botón "Manual".
      • Aparecerán botones para "Encender" y "Apagar" el motor.
      • Presiona "Encender" para activar el riego.
      • Presiona "Apagar" cuando ya no sea necesario regar.

      Solución de Problemas

        ❌ No se conecta el Bluetooth:

      • Asegúrate de que el módulo de riego esté encendido.
      • Verifica que el Bluetooth del celular esté activado.
      • Prueba desconectar y volver a conectar el módulo.

      ❌ El motor no enciende en automático:

    • Revisa si el umbral de humedad está correctamente configurado.
    • Asegúrate de que el motor esté funcionando manualmente.

    GardenPlus es un sistema de riego automático y manual basado en Arduino, que permite controlar el riego de tus plantas mediante una app móvil.

    Debes conectar VCC a 5V del Arduino, GND a GND, TX al pin 10 y RX al pin 11 con un divisor de voltaje de 1K y 2K.

    Se recomienda usar una bomba de 5V o 12V, y si es de 12V usar un relé y fuente externa para no dañar el Arduino.

    Necesitas un sensor de humedad del suelo (FC-28 o Hygrometer) y opcionalmente un sensor de nivel de agua.

    Descarga la app desde el repositorio de GitHub, instálala en tu dispositivo Android y habilita Bluetooth.

    En la app puedes alternar entre modo manual y automático. En manual controlas la bomba directamente, en automático el Arduino regula según la humedad del suelo.

    El Arduino funciona con 5V a 9V, y la bomba puede ser de 5V o 12V. Si usas 12V, conecta la bomba a una fuente externa mediante un relé.

    Sí, puedes agregar más sensores y bombas controladas por relés adicionales, adaptando el código de Arduino según sea necesario.
